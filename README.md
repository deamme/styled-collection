# Styled Collection
A collection of React components that uses `styled-components` with `styled-system` and `react-spring`.

## Commands
### Linting
Checking linting errors and warnings:

`npm run lint`

Fixing lints with prettier:

`npm run lint:fix`

### Docz
Development mode:

`npm run docz:dev`

Production build:

`npm run docz:build`
